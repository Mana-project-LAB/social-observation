# social-observation
This repository deconstructs various phenomena in the real world, performs fragment integration reasoning with seemingly unrelated events, and conducts analysis that goes beyond commonly accepted facts.
Furthermore, since the possibility of correction exists even for the facts used as reference material, the descriptions are strictly hypotheses. While hypotheses, we aim for a quality that cannot be dismissed as mere conspiracy theory, supported by as much circumstantial evidence and insight as possible.


このリポジトリでは実社会におけるさまざまな事象を解体し、一見関係なく見える別の事象と断片統合推論を行い、一般的に認知されている事実から一歩踏み込んだ考察を行う。
なお、参考資料とする事実に対しても訂正の可能性を考慮するため、記述はあくまでも仮説である。仮説ではあるが、可能な限りの状況証拠と洞察をもって、陰謀論と一笑に付すのは難しいクオリティを目指すつもりである。
